Tratamento de dados

1. Guardando resultado em variável

Para que um valor ou string recebido por meio do usuário através do comando prompt seja perdido, é necessário guardá-lo em uma variável.
Portanto, no caso de o usuário precisar digitar seu nome:

val nome = window.prompt('Digite seu nome')

2. Formatando Strings

Para realizar uma operação entre valores recebidos por um usuário através do comando prompt, é necessário realizar uma conversão. Isso ocorre pois o comando 'prompt' retorna apenas dados do tipo string.

A conversão pode ser feita antes da operação, diretamente nos valores recebidos n1 e n2, ou então na variavel final, atravéz do comando Number(valor).
Exemplo:

var n1 = Number(window.prompt('Digite um número:'))
var n2 = Number(window.prompt('Digite outro número:'))

O template string é um tipo de formatação de string que deixa o código bem mais elegante e organizado. Ou seja, é possível utilizar vários comandos para formatação de strings:
Seja var s = 'Bea', temos:

  • Não faz interpolação: 'Meu nome é s'
  • Usa concatenação: 'Meu nome é ' + s
  • Usa template string: `Meu nome é ${s}`

    • Outras funcionalidades com strings

    Existem inúmeras funções interessantes para se usar com strings, entre elas:

  • Contar nº de caracteres: s.length
  • Colocar tudo em MAIÚSCULO: s.toUpperCase()
  • Colocar tudo em minúsculo: s.LowerCase()

    • 3. Formatando números

    Assim como as strings, existem diversos comandos para formatação de variáveis numéricas.
    Seja var n = 1545.5, temos:

  • Altera número de casas após a vírgula: n.toFixed(número de casas)
